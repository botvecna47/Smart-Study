Java Learning Roadmaps

Explore three comprehensive 12-week roadmaps: Core Java, Java DSA, and Java UI/UX. Expand each section and week to see resources and practice questions.

Core Java (Weeks 1–12)

Week 1: Java Introduction & Setup
Topics Covered
  • What is Java? History and features
  • Install JDK & IntelliJ/Eclipse
  • First Java Program: Hello World
  • Keywords, Identifiers, Comments
Practice Questions
  1. Easy: Print 'Hello World' in Java
  2. Easy: Print the sum of two numbers
  3. Medium: Divide two numbers and print the result
  4. Medium: Print the result of an arithmetic expression
  5. Hard: Compute the specified expressions and print the output
Week 2: Variables, Data Types, and Operators
Topics Covered
  • Primitive & Non-primitive types
  • Type casting, literals
  • Arithmetic, relational, logical, bitwise operators
  • Expressions & precedence
Practice Questions
  1. Easy: Print the product of two numbers
  2. Easy: Print the sum, difference, product, and average of two numbers
  3. Medium: Print the remainder of two numbers
  4. Medium: Swap two variables
  5. Hard: Multiply two binary numbers
Week 3: Control Statements
Topics Covered
  • if, if-else, nested if
  • switch-case
  • Loops: for, while, do-while
  • Break, continue, labels
Practice Questions
  1. Easy: Check if a number is positive or negative
  2. Easy: Check if a number is even or odd
  3. Medium: Find the largest among three numbers
  4. Medium: Print the multiplication table of a number
  5. Hard: Print all prime numbers between 1 and 100
Week 4: Arrays & Strings
Topics Covered
  • 1D, 2D arrays, array methods
  • String handling, String methods
  • StringBuilder and StringBuffer
Practice Questions
  1. Easy: Sum values of an array
  2. Easy: Find the index of an array element
  3. Medium: Remove a specific element from an array
  4. Medium: Copy an array by iterating the array
  5. Hard: Find the maximum and minimum value of an array
Week 5: Functions and Methods
Topics Covered
  • Defining and calling methods
  • Method overloading
  • Call by value/reference
  • Recursion basics
Practice Questions
  1. Easy: Print the sum of two numbers using method
  2. Easy: Find the average of three numbers using method
  3. Medium: Check if a number is even or odd using method
  4. Medium: Find the factorial of a number using recursion
  5. Hard: Find the GCD of two numbers using recursion
Week 6: Object-Oriented Programming - Part 1
Topics Covered
  • Classes and objects
  • Constructors and types
  • this keyword
  • Access modifiers
Practice Questions
  1. Easy: Create a class and object in Java
  2. Easy: Create a class with a method and call it
  3. Medium: Create a class with a constructor and initialize values
  4. Medium: Create a class with private variables and public methods
  5. Hard: Create a class with static and instance variables
Week 7: Object-Oriented Programming - Part 2
Topics Covered
  • Inheritance and super keyword
  • Method overriding
  • Polymorphism (compile-time & runtime)
Practice Questions
  1. Easy: Create a subclass and call the superclass method
  2. Easy: Demonstrate method overriding
  3. Medium: Use super keyword to access superclass variables
  4. Medium: Demonstrate runtime polymorphism
  5. Hard: Create a multilevel inheritance example
Week 8: Abstraction & Encapsulation
Topics Covered
  • Abstract classes and interfaces
  • Encapsulation and getters/setters
  • Static and final keywords
Practice Questions
  1. Easy: Create an abstract class and subclass
  2. Easy: Implement an interface in a class
  3. Medium: Use abstract methods in subclasses
  4. Medium: Demonstrate encapsulation with private variables
  5. Hard: Use static and final keywords in a class
Week 9: Exception Handling
Topics Covered
  • try, catch, finally
  • Multiple catch blocks
  • throw and throws
  • Custom exceptions
Practice Questions
  1. Easy: Handle ArrayIndexOutOfBoundsException
  2. Easy: Handle NumberFormatException
  3. Medium: Use try-catch-finally block
  4. Medium: Create a custom exception
  5. Hard: Throw and catch multiple exceptions
Week 10: Java Collections Basics
Topics Covered
  • Arrays vs Collections
  • ArrayList, LinkedList, HashSet, HashMap
  • Iterators and loops
Practice Questions
  1. Easy: Create an ArrayList and add elements
  2. Easy: Iterate through all elements in a LinkedList
  3. Medium: Remove an element from a HashSet
  4. Medium: Get the number of elements in a HashMap
  5. Hard: Sort elements in an ArrayList
Week 11: File Handling
Topics Covered
  • FileReader, FileWriter
  • BufferedReader, BufferedWriter
  • Read/write from/to files
Practice Questions
  1. Easy: Create a new file and write to it
  2. Easy: Read a file and display its contents
  3. Medium: Append text to an existing file
  4. Medium: Count the number of lines in a file
  5. Hard: Find the longest word in a file
Week 12: Mini Project Week
Project Options
  • Student Record Manager
  • Library Book System
  • Contact Manager App
Practice Questions
  1. Easy: Student Record Manager
  2. Easy: Library Book System
  3. Medium: Contact Manager App
  4. Medium: Inventory Management System
  5. Hard: Banking System
